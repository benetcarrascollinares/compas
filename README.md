# 🎵 Compás

A multiplayer rhythm game where players compete online, create beatmaps and share songs with the community.

🌐 **Play online:** https://compas-production-a237.up.railway.app/

## ✨ Features

- 🎵 Real-time multiplayer rhythm battles
- 🏆 Online tournaments (4 or 8 players)
- 👁️ Live spectator mode
- 🛠️ Beatmap editor
- 🎶 Community-created songs
- 🔄 Match replay system
- ⚡ Energy system and special attacks
- 🛡️ Shield mechanics
- 💥 Battle effects: Flash, Shake, Reverse
- 📊 ELO rankings and player profiles
- 🏅 Unlockable titles and note skins
- 📱 PWA — installable on mobile and desktop

## 🧱 Tech stack

**Frontend:** HTML · CSS · JavaScript (ES Modules)

**Backend:** Node.js · Express · Socket.IO

**Database:** PostgreSQL

## 🎮 How to play

1. Open the game and create an account
2. Choose a song and difficulty
3. Wait for an opponent
4. Hit notes with: `D` `F` `J` `K`

**Special abilities:**
- `SPACE` → Special attack
- `SHIFT` → Shield
- `Q` → Flash
- `E` → Shake
- `R` → Reverse

## 🛠️ Local development

```bash
git clone https://github.com/benetcarrascollinares/compas.git
cd compas
npm install
node server/server.js
```

Open: http://localhost:3000
