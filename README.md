# 🎵 Compás

A multiplayer rhythm game where players compete online, create beatmaps and share songs with the community.

🌐 Play online:
https://compas-production-a237.up.railway.app/


## ✨ Features

- 🎵 Real-time multiplayer rhythm battles
- ⚡ Energy system and special attacks
- 🛡️ Shield mechanics
- 💥 Battle effects:
  - Flash
  - Shake
  - Reverse controls

- 🏆 Online tournaments
- 📊 Rankings
- 🛠️ Beatmap editor
- 🎶 Community-created songs
- 👁️ Spectator mode


## 🧱 Tech stack

Frontend:
- HTML
- CSS
- JavaScript (ES Modules)

Backend:
- Node.js
- Express
- Socket.IO

Database:
- SQLite (current)
- PostgreSQL migration planned


## 🎮 How to play

1. Open the game
2. Choose difficulty
3. Wait for an opponent
4. Hit notes with:

D F J K


Special abilities:

SPACE → Special attack

SHIFT → Shield

Q → Flash

S → Shake

R → Reverse


## 🛠️ Local development

Clone:

git clone https://github.com/YOUR_USER/compas.git

Install backend dependencies:

npm install

Start server:

node server.js


Open:

http://localhost:3000
